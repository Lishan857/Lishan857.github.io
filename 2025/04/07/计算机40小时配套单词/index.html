计算机的基本元件 - 开关

开关——switch
继电器——relay (electronics)
二极管——diode
真空管——vacuum tube
晶体管——transistor

布尔逻辑和逻辑门

布尔逻辑——Boolean logic
逻辑门——logic gate
与门——AND
或们——OR
非门——NOT
异或门——XOR

二进制

二进制——system binary
码制——code system

算数逻辑单元 ALU

算数逻辑单元 ALU | Arithmetic and Logic Unit
全加器FA (Full Adder)
半加器HA (Half Adder)
溢出——StackOverFlow
加法——ADD
带进位的加法——ADD WITH CARRY
减法——SUBTRACT
带借位的减法——SUBTRACT WITH BORROW
增量——INCREMENT
减量——DECREMENT
数字无改变通过——PASS THROUGH
Stack Overflow——溢出标志，操作溢出
Zero——为 0 标志，通过一堆或门对每个数字进行判断得出是否全部位为 0
negative——为负标志，相减看是否大于 0 判断

寄存器和内存

随机存储器 (RAM) ——random access memory
持久存储——persistent store/memory
锁存器——LATCH
AND-OR 锁存器 (AND-OR LATCH)
门锁——GATED LATCH
允许写入线——WRITE ENABLE
数据输入——DATE INPUT
数据输出——DATE OUTPUT
寄存器——register
位宽——width
矩阵——matrix
允许读取线——read enable
多路复用器——MULTIPLEXER

中央处理器 CPU

CPU——Central Processing Unit
指令寄存器——INSTRUCTION REGISTER
指令地址寄存器——INST.ADDR.REGISTER
取指令——fetch
解码——decode
执行——execute
时钟速度——clock speed

指令和程序

可编程的——programmable
硬件——hardware
软件——software

高级 CPU 设计

GHz——Gigaherz
总线——bus
缓存——cache
缓存命中——cache hit
缓存未命中/缺失率——cache miss
脏位 ——dirty bit
指令流水线——instruction pipelining
并行处理——parallelize
乱序执行——out-of-order execution
条件跳转——conditional jump instructions
推测执行——speculative execution
清空流水线——pipeline flush
分支预测——branch prediction
超标量处理器——superscalar
双核处理器——dual core processors
四核处理器——quad core processors
超级计算机——supercomputer
每秒浮点运算次数 (FLOPS)——FLOATING POINT MATH OPERATIONS PER SECOND

早期编程方式

插线板——plug boards
存储程序计算机——Stored-program Computers
冯诺依曼结构——Von Neumann Architecture
穿孔纸卡——punch cards
纸带——punched paper tape
面板编程——PANEL PROGRAMMING

编程语言发展史

机器语言/机器码——Machine Language / Machine Code
伪代码——Pseudo-Code
助记符——mnemonics
汇编语言——Assembly Language
算术语言版本0（A-0）——Arithmetic Language Version
变量——variables
FORTRAN——formula translation
普通面向商业语言口(COBOL)——Common Business-Oriented Language

编程原理 - 语句和函数

赋值语句——ASSIGHMEHT STATEMENT
初始化变量——initialize variables
控制流语句——Control Flow Statements
条件语句——Conditional Statements
While 语句——while loop
For 语句——for loop
方法/函数/子程序——functions/methods/subroutines

算法入门

算法——ALGORITHM
数组/数组/向量——Array/List/Vector
输入大小——input size
运算步数——the number of steps the algorithm
算法的复杂度——the complexity of the Selection Sortalgorithm
选择排序 O(n^2)——Selection sort
归并排序 O(n * log n)——Merge sort
图搜索——GRAPH SEARCH

数据结构——DATA STRUCTURES

下标——index
字符串——string
连接字符串——strcat
结构体——struct
节点——node
指针——pointer
链表——linked list
循环链表——circular
队列——queue
栈——stack
入栈——push
出栈——pop
树——tree
根节点——root
子节点——children nodes
母节点——parent nodes
叶节点——leaf
二叉树——binary tree
图——graph data
红黑树——red-black trees
堆——heaps

图灵

图灵机——TURING MACHINE
图灵完备——Turing Complete
图灵测试——Turing test
图灵奖——Turing Award

软件工程

层级——hierarchies
对象——object
面对对象编程（OO）——Object Oriented Programming
程序编程接口（API）——Application Programming Interface
集成开发环境（IDE）——Integrated Development Environments
调试——debug
版本控制——version control /revision control
代码仓库——code repository
回滚——ROLLED BACK
质量保证测试（QA）——Quality Assurance testing

集成电路与摩尔定律

分立元件——discrete components
集成电路——Integrated Circuits
印刷电路板——printed circuit boards
光刻——Photolithography
半导体——semiconductor
晶圆——wafer
光刻胶——photoresist
光掩膜——photomask
氧化层——oxide layer
掺杂——doping
双极型晶体管——bipolar junction transistor
微型处理器——microprocessor
超大规模集成软件——very-large-scale integration/VLSl software

操作系统——Operating systems

批处理——batch processing
设备驱动程序——device drivers
输入输出设备（I/O）——Input & Output
多任务处理——MULTITASKING
虚拟内存——virtual memory
动态内存分布——dynamic memory allocation
内存保护——Memory Protection
分时操作系统——time-sharing
内核——kernel

内存和储存介质

存储器——storage
延迟线/顺序/循环存储器——Delay Line Memory / sequential / cyclic-access memory
随机存取储存器——random access memory
磁芯存储器——MAGNETIC CORE MEMORY
磁带——magnetic tape
磁鼓存储器——Magnetic Drum Memory
磁盘——Hard Disk Drives
寻道时间——the seek time
软盘——floppy disk
光盘——Compact Disk
固态——solid state

文件与文件系统

文件格式——file format
元数据（数据的数据）——meta data
文件头——Header
位图（.BMP）——Bitmap
文件系统——File System
平面文件系统——flat file system
分层文件系统——hierarchical file system
根目录——Root Directory

压缩——COMPRESSION

游程编码——Run-Length Encoding
霍曼夫数——Huffman Tree
时间冗余——temporal redundancy

命令行界面 | 人机交互——Human-Computer Interaction

命令行界面——command line interfaces
终端——terminals

屏幕和 2D 图形显示

阴极射线管（CRT）——Cathode Ray Tubes
矢量扫描——Vector Scanning
光栅扫描——Raster Scanning
像素——pixel
液晶显示器（LCD）——Liquid Crystal Displays
字符生成器——character generator
只读存储器（ROM）——Read only memory
屏幕缓存区——Screen buffer
计算机辅助设计（CAD）——Computer-Aided Design

个人计算机革命

图形用户界面 Graphical User Interface

3D 图形

线框渲染——Wireframe Rendering
正交投影——ORTHOGRAPHIC PROJECTION
透视投影——PERSPECTIUE PROJECTION
网格——mesh
扫描线渣染——Scanline Rendering

局域网——Local Area Networks
广域网(WAN)——WIDE AREA NETWORK

媒体访问控制地址（MAC）——Media Access Control address

互联网

互联网协议（IP）——Internet protocol

端口号——port number
传输控制协议（TCP）——Transmission Control Protocol

万维网 WWW World Wide Web

超链接——hyperlinks
